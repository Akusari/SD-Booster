SD-Booster version 2.x FAQ

Q: What's the best cache value setting for my smartphone ?

A: Well, i don't have the perfect answer for you. It depends on your smartphone, your SD-Card(s) and the data (files) stored on your SD-Card(s).
It's possible that your phone support and use more than one card. In this case we have an internal and external card. Both cards using different filesystems with different requirements for the cache size. SD-Booster is the only app which shows each card separately!

There are some values based on experience by other users and my own tests.
Each card has it's own item. The "Cache:  <size>" text line gives you a little hint which value might be good for you. The terms are color based:
  • Red: Cache size is to small
  • Yellow: Cache size is okay, but could be bigger
  • Green: Cache size seems optimal
  • Magenta: Cache size seems to big.
The calculation only based on intersection values (experience) which might doesn't fit in your case.

Basicly you can use my standard suggestion of 1024KB and/up to 2048KB. It should be works and speedup your device anyway. If you wanna have the better boost for your phone, you need to do some speed tests with the „SD Tools“.  Just setup SD-Booster with different values like 512, 1024 or 2048KB, apply and start „SD Tools“ to compare your results. Please note that such benchmark tools like SD-Tools never can simulate daily business like starting a app or load database data. It's only a apporoximate value.

 It's rarly possible that you find your best value, but your device seems "slow down" at all. Please reduce your values to 512 or less in such cases.

PS: SD-Booster version 2.1 will be shipped with his own speed-test!

Q: How do I get root permission ?

A: First over all, not every phone is rootable yet and it might takes some time to get it . The best way to know if your device is rootable, use Google in order to help you. Just start a Google search with your phone model and the keyword "root".

Q: What means "Monitor" all or one device ?

A: SD-Booster checks every configured device or all devices if the settings of this device seems okay. If SD-Booster founds a problem, it will be fixed and reported through notification.
Q: What means "Notification on state change" ?

A: SD-Booster gets a notification if a SD-Card is removed or inserted physically or just mounted from Android. In this case the (new) device will be checked  if it's managed by SD-Booster or not. Depends on this a notification will be created if it's new or the cache size set successful.
Q: SD-Booster doesn't set my value on system start

A: Please be sure that you first enable "Cache for all devices" and set a value or setup each card separately with the setup button. In the setup dialog for each card you need to enable "Set this device on boot".

Q: Is it possible that SD-Booster can destroy any data ?

A: Definitely no, because SD-Booster doesn't work with the data and files itself. SD-Booster just prepare a special memory pool where used files will be cached for a faster access.
